# Inventory Management System

A Full Stack Inventory Management System built using React.js, Node.js, Express.js, and PostgreSQL.

## Features

### Dashboard

* Total Products Count
* Total Categories Count
* Total Brands Count
* Total Suppliers Count
* Low Stock Products Count

### Category Management

* Add Category
* Edit Category
* Delete Category
* Search Categories

### Brand Management

* Add Brand
* Edit Brand
* Delete Brand
* Search Brands

### Supplier Management

* Add Supplier
* Edit Supplier
* Delete Supplier
* Search Suppliers

### Product Management

* Add Product
* Edit Product
* Delete Product
* Search Products
* Brand & Category Mapping
* Minimum Stock Tracking

### Inventory Management

* Stock In
* Stock Out
* Inventory Quantity Updates
* Stock History Tracking

### Low Stock Monitoring

* Identify products below minimum stock level
